# GDT Daily

A daily microlearning broadcast for **ASME Y14.5-2018** geometric dimensioning and tolerancing.

One lesson. One diagram. Every morning, 07:00 ET.

**Live:** [vishutdhar.github.io/gdt-daily](https://vishutdhar.github.io/gdt-daily/)

Each lesson is a single SVG diagram and a few sentences on what the symbol controls, how to read it, and the mistake people make their first time through.
